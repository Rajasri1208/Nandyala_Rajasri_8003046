## Description

This repository contains exercises, assignments, hands-on tasks, and solutions completed as part of the **Cognizant Digital Nurture 5.0 Java Full Stack Engineer (FSE) Upskilling Program**.

The repository demonstrates practical implementation of front-end technologies, database concepts, SQL programming, Core Java, Object-Oriented Programming principles, exception handling, file handling, collections, JDBC connectivity, multithreading, networking, and modern Java features.
